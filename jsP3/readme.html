String Methods

Methods actions that can be performed on objects.
Formats:-
stringName.method()

str.trim()

Stirngs are Immutable in JS

No changes can be made to strings
Whenever we do try to maje a change, a new string is created and old one remains same.

Upper case = toUpperCase(), Lower case = toLowerCase()

String Methods with Arguments

Argument is a some value that we pass to the method
Format:- stringName.method(arg)

indexOf

Returns the first index if occurrence of some value in string. Or gives -1 if not found.
let str = "ILoveCoding"
str.indexOf("Love")

Method Chaining

Using one method after another. Order of execution will be left to right.
str.toUpperCase().trim()

Slice

Returns a part of the original string as a new string.
Syntax :- str.slice(start, end)
let str = "ILoveCoding"
str.slice(5)
str.slice(1,4)
str.slice(-num)

Replace

Searches a value in the string & returns a new string with the value replaced.
let str = "ILoveCoding"
str.replace("Love", "Do")

Repeat

Returns a string with the nimber of copies of a string.
let str = "Mango"
str.repeat(3)

Array(Data Structure)

Linear collection of things

Arrays are mutable

Array Method :-

push: add to end
unshift: add to start
pop: delete from end & returns it
shift: delete from start & returns it
indexOf
includes
concat
reverse
slice :- copies a portion of an array
splice :- removes/ replaces/ add elements in place (Syntax:- splice(start, deleteCount, item0...itemN))
sort :- sorts an array

Reference (bole to address in memory)

Nested Arrays(Multi- dimensional array) :- array of arrays
