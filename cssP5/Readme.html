What is Flexbox?

Flexible Box Layout
It is a one-dimensional layout method for arranging items in rows or columns.

Flexbox direction

It Sets how flex items are placed in the flex contianer, along which axis and direction.
(i) flexbox-direction: row; (main axis, left to right)
(ii) flexbox-direction: row-reverse; (main axis, right to left)
(iii) flexbox-direction: column; (main axis, top to bottom)
(iv) flexbox-direction: column-reverse; (main axis, bottom to top)
