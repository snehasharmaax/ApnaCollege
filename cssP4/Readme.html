CSS Transform

This property lets you rotate, scale, skew, or translate an element.
(i) Rotate :- transform: rotate (45deg);
(ii) Scale :- transform: scale(1.5)
(iii) Translate :- transform: translate(50px, 50px), translateX(10px), translateY(10px)
(iv) Skew :- transform: skew(30deg)

Position

The position CSS property sets how an elementis positioned in a document.
The top, right, bottom, and left properties determine the funal location of positional elements.

  • Static :- The TRBL and z-index properties have no effect.
    This is the default value
  • Relative :- The offset is relative to itself based on the values of toop, right, bottom and left.
  • Absolute :- The element is removed from the normal document flow, and no space is created for the element in the page layout.
    It is positioned relative to its closest positioned ancestor, if any: otherwise, it is placed relative to the i initial containing block.
  • Fixed :- The element is removed from the normal document floe, and no space is created fot the element in the page layout.
    It is positioned relative to the initial containing block established by the viewport.
