Alpha Channel

Sets the opacity for a color.
Ranges from 0 to 1
rgba(255, 255, 255, 0.3 )

Opacity

Sets the opacity for the element
Ranges from 0 to 1
opacity: 0.5

CSS Transitions

Trnasitions enable you to define the transition between two states of an element.

transition Shorthand
property name duration timing-function delay

CSS Transform

This property lets you rotate, scale, skew, or translate an element.
(i) Rotate :- transform: rotate (45deg);
(ii) Scale :- transform: scale(1.5)
(iii) Translate :- transform: translate(50px, 50px), translateX(10px), translateY(10px)
(iv) Skew :- transform: skew(30deg)

Box Shadow

It adds shadow effects around an element's frame
Eg. - box-shadow:2px 2px 10px green;

Background Image

Lets you set an image as a background.
Eg- background-image:url("abc.jpg")
background-size:contain / cover / auto;
