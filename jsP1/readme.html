The NaN global property is a value representing Not-A-Number
  • 0 / 0
  • NaN - 1
  • NaN * 1
  • NaN * NaN

Operator Precedence

This is the general order of solving an expression.

  • ()
  • **
  • *, /, %
  • +, -
  • Eg:- ( 5 + 2 ) / 7 + 1 * 2
