CSS Grid

Setting a container's display to grid will make all children grid items.
Eg. container{ display: grid}

Grid Template

They define the lines & track sizing.
Eg. grid-template-rows: 50px 50px 50px 50px 50px;
grid-template-columns: 200px 200px 200px;

Grid Template (Repeat)

Repeat is used to divide all available space .
grid-template-rows: repeat(count, 1fr);
grid-template-columns: repeat(count, 1fr)

Eg. grid-template-rows: repeat(2, 1r)
grid-template-rows: 1fr 1fr
1fr = 1 fraction of 100% width

Grid Gaps

They define the gaps between the lines.
row-gap: 10px
column-gap: 10px
grid-gap : rowGap columnGap

Grid Columns

Defines the item's starting & ending posiiton inside the column.
grid-column-start: line-number
grid-column-end: line-number
grid-column: start-col/ end-col
grid-column: start-col/span-number

Grid Rows

Defines the item's starting & ending posiiton inside the row.
grid-row-start: line-number
grid-row-end: line-number
grid-row: start-row/ end-row
grid-row: start-row/span-number

CSS Animations

To animate CSS elements
Syntax:- @keyframe myName{ from {font-size: 20px} (initial)
to {font-size: 40px} (final) }

  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction

% in Animation

Syntax :- @keyframes myName{
0%{font-size:20px}
50%{font-size:30px}
100%{font-size:40px}
}

Media Queries

Help create a responsive website.
media features - width (of viewport)
Eg. @media(width: 400px){
    div{
      background-color: red;
    }
}
