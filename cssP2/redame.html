Selectors in CSS

Universal Selector

To select everything in a document. (*)



Element Selector

Selects all elements of the same type. (h1)



ID Selector

Selects an element based on the value of the element's id attribute. (#idname).



Class Selector

Selects an element based on their class attribute. (.className)



Descendent Selector

Eg - Selects all paragraphs inside divs.



Adjacent Sibling Combinator

Eg - Heading3 that comes immediately after any paragraph. (p+h3)



Child Combinator

Eg - Selects all buttons which are direct children of spans. (parent > child)



Attribute Selector

Selects elements based on the presence or value of a given attribute.



Psuedo Class

A keyword added to selector that specifies a special state of the seleceted element.

:hover :active :checked :nth-of-type
